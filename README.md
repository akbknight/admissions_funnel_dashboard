# Kogod Admissions Funnel Dashboard

## Overview
This interactive dashboard provides a high-density, executive-level view of the admissions funnel, comparing Fall 2026 vs Fall 2025 performance.

## Features
- **Smart Filtering**: Automatically switches between Tab view (All Programs) and Single Program view.
- **Precision Date Logic**: YoY comparisons based on strict file-date anchors.
- **High-Density UI**: Custom HTML grid for maximum data visibility without scrolling.
- **Executive Branding**: Kogod/American University styling.

## How to Run locally
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Deployment
This app is ready for **Streamlit Cloud**.
1. Push this repo to GitHub.
2. Go to [share.streamlit.io](https://share.streamlit.io).
3. Select this repository and the main file `app.py`.
